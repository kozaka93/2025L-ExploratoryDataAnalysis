Najwieksze saldo a koncie posiadaja osoby o najwyzszym poziomie wyksztalcenie, natomiast najmniejsze odpowiednio osoby o srednim, a w drugiej kolejnosci podstawowym wyksztalceniu.Najwiekszy odstetek niesplacanych kredytow jest w grupie osob, ktorych wyksztalcenia nie znamy, natomiast najmniejszy w grupie z wyksztalceniem wyzszym. ## Zadanie 4 Czy dzień i miesiąc kontaktu z klientem ma wpływ na średni czas trwania rozmowy w stosunku do średniego czasu trwania rozmowy? ```{r zadanie 4, message=FALSE} overall_avg_duration <- mean(df$duration, na.rm = TRUE) daily_duration <- df %>% group_by(day) %>% summarise(avg_duration = mean(duration, na.rm = TRUE)) monthly_duration <- df %>% group_by(month) %>% summarise(avg_duration = mean(duration, na.rm = TRUE)) daily_duration <- daily_duration %>% mutate(day = as.character(day)) monthly_duration <- monthly_duration %>% mutate(month = as.character(month)) combined_duration <- daily_duration %>% mutate(type = "Dzień miesiąca") %>% rename(time_unit = day) %>% bind_rows( monthly_duration %>% mutate(type = "Miesiąc") %>% rename(time_unit = month) ) ggplot(combined_duration, aes(x = time_unit, y = avg_duration, fill = type)) + geom_col() + geom_hline(yintercept = overall_avg_duration, linetype = "dashed", color = "red") + facet_wrap(~type, scales = "free_x") + labs( title = "Średni czas rozmowy w zależności od dnia i miesiąca", x = "Jednostka czasu: dzień/miesiąc", y = "Średni czas rozmowy (sekundy)" ) + theme_minimal() + scale_fill_manual(values = c("steelblue", "coral")) ``` **Komentarz:** Najcześciej rozmowy trwaja dłużej w grudniu, oraz w dalszej kolejności w kwietniu, wrzesniu i pazdzierniku.
